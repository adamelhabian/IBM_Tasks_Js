# Healthcare Data Analysis Interface

## Project Overview

This project is a simple healthcare data analysis web application built using HTML, CSS, and JavaScript. It allows users to add patient information, analyze healthcare data, and search for medical conditions dynamically.

The project focuses on DOM manipulation, event handling, data processing, and dynamic report generation.

---

# Features

- Add patient information
- Store data using arrays and objects
- Generate healthcare reports dynamically
- Search for health conditions using JSON data
- Display symptoms, prevention, treatment, and images
- Responsive and simple user interface

---

# Technologies Used

- HTML5
- CSS3
- JavaScript
- JSON
- Fetch API

---

# How It Works

## Add Patient

Users enter:

- Name
- Gender
- Age
- Health condition

JavaScript validates the inputs, stores the patient data, and updates the report automatically.

---

## Generate Report

The application dynamically displays:

- Total number of patients
- Conditions breakdown
- Gender-based statistics

---

## Search Condition

The search feature uses `fetch()` to read data from a JSON file and display:

- Condition name
- Symptoms
- Prevention methods
- Treatment
- Related image
